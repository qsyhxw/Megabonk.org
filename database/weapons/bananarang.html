Bananarang flies out in a straight line, damaging all enemies in its path. After reaching maximum range, it returns to the player, hitting enemies again.
Throw Range: Medium to long distance
Return Damage: Full damage on return trip
Hit Pattern: Line projectile both ways
üîÑ Auto-Return
Unlike other projectiles, Bananarang automatically returns to the player's position. This creates a unique double-hit mechanic for positioned enemies.

Homing: Returns to current player position
Double Hit: Enemies can be hit twice per throw
üéØ Precision Targeting
Bananarang aims toward the nearest enemy or highest threat target. Combined with return mechanics, it provides excellent area control.
Targeting: Smart enemy prioritization

Positioning: Player movement affects return path
