Quick Answer

Wisps are a Desert enemy. For the Dragon's Breath objective, play Fox on Desert and accumulate 1,000 Wisp kills. Tier 2 or Tier 3 is a sensible farming choice because community routes report Wisps there more consistently, but the achievement itself does not require a particular tier.

Confirm You Are Killing the Right Enemy

Do not rely on an enemy name you remember from another map or on a similar ghost-like silhouette. The most reliable identification method is a short controlled test:

  1. Record the current objective count

    Check the Dragon's Breath or Wisp-related counter before the run.

  2. Enter Desert as Fox

    Keep both required conditions fixed so the test has only one unknown: the enemy identity.

  3. Kill a visible group

    Continue only long enough to create a clear before-and-after comparison.

  4. Check the counter again

    If it moved, you have confirmed the correct target and can commit to the farming route.

Recommended Wisp Farming Route

  1. Select Fox

    The objective explicitly requires Wisp kills as Fox. A faster character does not help if the kills do not satisfy that condition.

  2. Choose Desert Tier 2 or Tier 3

    Start with the higher tier you can clear safely. Consistent survival and coverage are more useful than forcing a tier that overwhelms the build.

  3. Use Swarm only if the build can clear it

    Extra enemy pressure can create more farming opportunities, but it also makes weak coverage fail faster. Treat Swarm as an optional accelerator.

  4. Confirm Wisps before committing

    Use the enemy mix and a short progress-counter test. Restart an unproductive run early.

  5. Keep broad damage online

    Prioritize coverage that continues working while Fox repositions. Do not let single-target upgrades consume the entire build.

  6. Repeat until the counter reaches 1,000

    The objective can be accumulated across qualifying runs, so protect good progress instead of gambling a run on unnecessary risks.

Why Wisp Kills Are Not Counting

Wrong character

The Dragon's Breath condition says as Fox. Wisp kills made with another character do not satisfy that wording.

Wrong map

The condition says on Desert. Move the farm to Desert before diagnosing the enemy itself.

Wrong enemy

Run a short before-and-after counter test. If the value stays unchanged with Fox on Desert, do not keep farming that same target.

Unproductive spawn mix

Not every run is equally useful. Confirm Wisps are present and restart an attempt that never develops into a Wisp farm.

Too much pressure

Swarm or a high tier can reduce actual kills if Fox spends the run retreating. Lower the pressure and restore reliable clear uptime.

Counter checked too early

Compare the objective count at a stable menu checkpoint. The tracker on this page is manual and does not read game data.

Wisp FAQ

Where do Wisps spawn in Megabonk?

Wisps are tied to the Desert map. Tier 2 and Tier 3 are commonly used for farming, but the Dragon's Breath achievement does not name a required tier.

Do I need to play as Fox?

Yes, if your goal is the Dragon's Breath achievement. Its condition explicitly says to kill the Wisps as Fox on Desert.

Do the 1,000 kills need to happen in one run?

No. The Wisp objective accumulates across qualifying Fox runs on Desert.

Are Wisps guaranteed in every Desert run?

Do not assume every run will provide the same enemy mix. Confirm that Wisps are present before committing to a long farm.

Should I use Swarm?

Swarm can increase farming opportunities, but only use it when your build can clear the added pressure. A stable standard run can outperform a Swarm run spent constantly retreating.

What is the fastest Wisp farming build?

There is no mandatory loadout. Start with broad coverage such as Aura or Lightning Staff, add enough damage to prevent stragglers, and protect Fox's movement uptime.
