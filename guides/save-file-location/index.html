Location, backup and safe restore

Megabonk Save File Location

Find the active save on Windows or Steam Deck, make a rollback-ready copy, and restore it without letting a Cloud conflict erase the version you meant to keep.

%USERPROFILE%\AppData\LocalLow\Ved\Megabonk
Quick answerLocationsFind active saveBack upRestoreSteam CloudEditing risksFAQ

Quick Answer

On Windows, press Win + R, paste %USERPROFILE%\AppData\LocalLow\Ved\Megabonk, and press Enter. Account data is below Saves\CloudDir\<Steam ID>. Close Megabonk before copying anything, then back up the entire Megabonk folder rather than guessing which single file matters.

Using Proton on Linux? Follow the same App ID and Windows-style path inside that library's Proton prefix. The library mount point may differ.

How to Identify the Active Save

1. Match the account

Check the numeric folder below Saves\CloudDir against the Steam account currently in use.

2. Compare timestamps

Close the game, then look for files updated after the most recent session.

3. Preserve before testing

Copy the whole root first. Never edit or overwrite the only candidate.

Do not test by deleting folders. Several Steam accounts, old Proton prefixes, or changed libraries can leave multiple plausible copies.

Make a Safe Manual Backup

  1. Close every writer

    Exit Megabonk and any mod manager. Wait for Steam synchronization to finish.

  2. Open the verified root

    Use the Windows or Proton path above. Confirm the account and recent timestamp.

  3. Copy the complete folder

    Copy Ved\Megabonk to a separate location, preferably another drive or private archive.

  4. Label the snapshot

    Use a dated name and record the game version, enabled mods, and Steam account.

  5. Keep milestone copies

    Retain a known-good vanilla copy plus pre-update and pre-mod copies.

Restore Without Losing the Better Copy

  1. Close Megabonk and Steam

    Do not replace files while the game or Cloud client may write them.

  2. Preserve the current state

    Copy or rename the current Megabonk folder to a dated holding folder.

  3. Control Cloud synchronization

    For a local-versus-Cloud conflict, preserve both copies, then temporarily turn off Steam Cloud before replacing files.

  4. Restore one matching snapshot

    Place the complete backup under AppData\LocalLow\Ved in the correct Windows profile or Proton prefix.

  5. Launch once and inspect

    Check unlocks, Silver, settings, and recent progression before accepting the restored state.

  6. Re-enable Cloud carefully

    When Steam presents a conflict, compare timestamps rather than selecting automatically.

Without a manual backup, the built-in Recover Save feature can rebuild achievement-linked unlocks. It may not restore Silver or progress that achievements do not track.

Steam Cloud Is Sync, Not Version History

What Cloud helps with

  • Moving the current save between devices
  • Keeping one active state synchronized
  • Recovering when one device still has the desired copy

What Cloud does not guarantee

  • A catalog of dated restore points
  • Protection from syncing damaged or edited data
  • Automatic selection of the better progress state

Conflict rule: preserve every available copy first. A newer timestamp does not necessarily mean healthier progress.

Updates and Save Compatibility

A newer build may migrate older save data. That does not prove the migrated copy can later open in an older build.

Keep the untouched original

Store a pre-update copy outside the live directory before major updates, Proton changes, or rollbacks.

Match the environment

Record game version, Proton version, enabled mods, and Steam account so failures can be isolated.

Save Editors and Full Unlock Saves: Know the Risk

This page does not provide an unlock-all or full-save download. Third-party data can contain invalid progression, disagree with achievements, break after an update, or invalidate fair leaderboard comparisons.

Before any edit

  • Keep an untouched original
  • Edit a disposable copy only
  • Match the exact game version
  • Test without other mods first

Stop immediately when

  • A download includes an unknown executable
  • Instructions request a Steam password
  • Files replace more than documented save data
  • Progress begins resetting or duplicating

For mod setup and compatibility profiles, use the Megabonk Mods Hub. Back up first; install second.

Megabonk Save File FAQ

Where is the Windows save?

Open %USERPROFILE%\AppData\LocalLow\Ved\Megabonk. Account data is below Saves\CloudDir\<Steam ID>.

Where is the Steam Deck save?

Use the compatdata/3405340 Proton prefix in the Steam library where the game is installed.

Why are there multiple Steam ID folders?

More than one account may have launched the game. Match the account and timestamps before restoring.

Does Steam Cloud replace a manual backup?

No. It synchronizes an active state but is not a dated archive.

Can I restore an old save after an update?

Possibly, but do not assume compatibility. Preserve the untouched original and test a copy.

Can I download a full unlock save here?

No. Unknown saves introduce compatibility, fairness, and security risks.
