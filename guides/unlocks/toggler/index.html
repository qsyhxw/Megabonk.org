Native reward-pool control

Megabonk Toggler Guide

Unlock Toggler, switch eligible content off before a run, and build a focused reward pool without downloading a mod or editing configuration files.

Toggler is a Shop Upgrade, not a Mod.
Quick answerUnlockHow to usePool strategyTroubleshootingFAQ

Quick Answer

Toggler is built into Megabonk. Purchase 40 unlocks to make the Shop Upgrade available, then buy it for 4 Silver. Open the Unlocks menu before a run and disable eligible content you do not want in future offer pools. There is no download, mod manager, keyboard shortcut or external config file.

How to Unlock Toggler

  1. Buy 40 unlocks.
    Progress comes from purchased unlock entries such as characters, weapons, Tomes and Items, not from opening 40 chests or merely finishing 40 quests.
  2. Return to the Shop.
    After the requirement registers, Toggler becomes an available Shop Upgrade.
  3. Spend 4 Silver.
    Buying the upgrade permanently adds its pool-control interface to your save.
  4. Open Unlocks before the next run.
    Review eligible entries and prepare the pool for the build you intend to play.

How to Use Toggler

1. Choose a build goal

Decide the character, primary weapons, Tome package and Item synergies before disabling anything.

2. Open the Unlocks menu

Use the game's own menu. Toggler is not activated from a BepInEx panel or keyboard hotkey.

3. Disable eligible misses

Switch off unlocked options that do not support the planned run. Some default or required starter content may remain available.

4. Start a new run

Toggler changes future offer pools. It does not replace a choice already on screen or remove rewards already collected.

What it does not do: Toggler does not improve rarity, turn weak Items into strong ones, guarantee one exact reward, or alter a character's required starting weapon.

Toggler vs Banish

ToolWhen usedPurposeBest use
TogglerBefore a run, in UnlocksRemoves eligible unlocked content from future pools until re-enabled.Prepare a consistent pool for a known build.
BanishDuring a runRemoves an offered option from that run's future choices.React to the actual offers and finish pruning.
Skip / RefreshOn an offer screenDeclines or rerolls the current selection.Recover when the remaining pool still produces a bad screen.

Do not over-filter: disabling content needed for an unlock challenge or a different character can make the next objective harder. Recheck Toggler whenever you change builds.

Toggler Troubleshooting

Not in the Shop

Confirm you purchased 40 actual unlock entries. Quest count, chest count and Silver balance are different counters.

Cannot toggle an entry

Only eligible content can be disabled. Default and required starter options may remain active.

A disabled option still appears

Start a fresh run and confirm the entry is still off. Existing offer screens and already collected rewards are not retroactively changed.

Build option disappeared

Return to Unlocks and re-enable it. Toggler is reversible; it does not delete the unlock.

No hotkey works

There is no separate Mod shortcut. Use the Unlocks menu interface.

Looking for config files

No external Toggler config is required. Mod manager profiles are unrelated to this native upgrade.

Toggler FAQ

Is Toggler a mod?

No. It is a native Shop Upgrade, so do not download an executable or third-party package claiming to be required.

How many unlocks do I need?

Purchase 40 unlocks. After that, buy Toggler for 4 Silver.

Does it disable Items, weapons and Tomes?

It controls eligible entries exposed by the Unlocks menu. Some defaults or required starters can remain available.

Does Toggler increase Luck or Item rarity?

No. It narrows eligible pools; it does not alter the rarity roll or Luck stat.

Can I undo a toggle?

Yes. Return to Unlocks and switch the entry back on before a future run.
