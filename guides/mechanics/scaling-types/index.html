Linear Scaling in Megabonk — How Additive Stacking Works

Linear Scaling (additive stacking) is the most common scaling type in Megabonk. Each additional copy of an item adds the same flat bonus — no diminishing returns. Understanding when and why items scale linearly is key to optimizing your builds.

📐 GAME MECHANICS • SCALING TYPES

📏 What Is Linear Scaling?

Linear Scaling is the simplest scaling mechanic in Megabonk. When an item uses linear scaling, every copy you pick up adds the exact same amount of bonus as the first. There are no diminishing returns — the 10th copy is just as valuable as the 1st.

Linear Scaling Formula
Total Bonus = Base Value + (Per-Copy Value × Number of Copies)
Each copy contributes the same fixed amount — predictable and straightforward.
💡 Example: Clover (+10% Luck)

Each Clover adds exactly +10% Luck. With linear scaling:

1 copy
+10%
2 copies
+20%
3 copies
+30%
5 copies
+50%
10 copies
+100%

The graph of linear scaling is a straight line — hence the name. Every step up the ladder is the same height.

💡 Bracket Tip

Items within the same damage bracket combine additively. But different brackets multiply with each other. This means having items in multiple brackets is more powerful than stacking everything in one bracket. See the Damage Brackets guide for details.

🔥 What Happens Above 100%?

For proc-based items (items that trigger an effect on hit), linear scaling can push the chance above 100%. When this happens, the item guarantees at least one proc and rolls for additional stacks.

💡 Example: Moldy Cheese (+40% Poison Chance)

Moldy Cheese adds +40% chance to poison enemies on hit. With additive stacking:

1 copy
40%
2 copies
80%
3 copies
120%
5 copies
200%

At 120%: guaranteed 1 poison stack + 20% chance for a 2nd stack.
At 200%: guaranteed 2 poison stacks per hit.
At 250%: guaranteed 2 stacks + 50% chance for a 3rd stack.

⚠️ Important: Proc Coefficient

Some items and weapons have a Proc Coefficient that affects the actual trigger chance. Items with a Proc Coefficient of 0.5 (like Cactus, Electric Plug, Holy Book) will halve the chance — so a 40% poison chance becomes 20% when triggered by these items. See the Proc Coefficient guide for the full list.

⚖️ Linear vs Hyperbolic Scaling

Not all items in Megabonk scale linearly. Some use Hyperbolic Scaling, which has diminishing returns. Understanding the difference is critical for build optimization.

📈

Linear Scaling

  • Same bonus every copy
  • No diminishing returns
  • Can exceed 100%
  • Predictable & consistent
  • Most common in Megabonk
📉

Hyperbolic Scaling

  • Each copy gives less than the last
  • Diminishing returns
  • Can never reach 100%
  • Used for balance
  • Examples: Key, Evasion stat
⚠️ Tricky Case: "Linearly Added, Hyperbolically Applied"

Some items add their raw value linearly (e.g., Slippery Ring adds +15% Evasion per copy), but the final stat uses a hyperbolic formula. This means:

  • 3 Slippery Rings → +45% raw Evasion → actual evasion is less than 45%
  • 10 Slippery Rings → +150% raw Evasion → actual evasion converges toward ~60%

The game's UI shows the raw stacked value, not the actual effective percentage. Always check if a stat uses hyperbolic conversion before over-investing in it.

🧠 Build Tips for Linear Scaling Items

1. Diversify Across Damage Brackets

Because items within the same bracket combine additively but brackets multiply together, having +50% from Bracket A and +50% from Bracket B gives 2.25× damage (1.5 × 1.5). Stacking +100% in a single bracket only gives 2.0× damage. Spread your damage sources across different brackets for maximum output.

2. Stack Early, Benefit All Game

Linear scaling items are especially valuable early in a run. Unlike hyperbolic items that lose value per copy, linear items maintain their full worth throughout. An early Clover or Gym Sauce will pay dividends for the entire run.

3. Over-100% Proc Stacking Is Powerful

For proc items like Moldy Cheese, pushing past 100% lets you apply multiple status effects per hit. This is a key strategy for poison and status-effect builds.

4. Watch Out for Items in Their Own Bracket

Some linear items like Battery (Attack Speed) and Time Bracelet (XP) exist in their own dedicated brackets. This means they multiply independently with your other bonuses, making them exceptionally efficient.

❓ Frequently Asked Questions

What is Linear Scaling in Megabonk?

Linear Scaling (also called additive stacking) means each additional copy of an item adds the same flat bonus. For example, each Clover adds +10% Luck, so 3 Clovers = +30% Luck. The formula is: Total Bonus = Base Value + (Per-Copy Value × Number of Copies).

Which items use Linear Scaling?

Most common items use Linear Scaling, including Clover (+10% Luck), Gym Sauce (+10% Damage), Oats (+25 Max HP), Battery (+8% Attack Speed), Time Bracelet (+8% XP), Turbo Socks (+15% Move Speed), Forbidden Juice (+10% Crit Chance), and many more. Check the table above for a full list.

What happens when Linear Scaling exceeds 100%?

For proc-based items like Moldy Cheese (+40% poison chance), going over 100% means you apply multiple stacks per hit. For example, 5 Moldy Cheeses = 200% chance = 2 guaranteed poison stacks per hit. The remainder above each 100% threshold rolls for an additional stack.

What is the difference between Linear and Hyperbolic Scaling?

Linear Scaling adds the same value every time with no diminishing returns. Hyperbolic Scaling has diminishing returns — each copy gives less benefit than the last. Evasion and some proc chances (like Key's free chest chance) use hyperbolic scaling to prevent reaching 100% too easily.

Why does Slippery Ring feel less effective at high stacks?

Slippery Ring adds +15% Evasion per copy linearly, but the Evasion stat itself uses hyperbolic scaling. So while the raw number goes up at the same rate, the actual dodge chance has diminishing returns. This is a common source of confusion — the item is linear, but the stat it feeds into is not.
