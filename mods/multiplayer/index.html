Quick Answer

Megabonk has no native co-op, but two active Thunderstore projects add unofficial multiplayer. MegabonkTogether is the broader netplay option with up to six players and private or random rooms. BonkWithFriends focuses on Steamworks friend invites and drop-in/drop-out P2P co-op.

Choose one, not both. The host and every client should run the same multiplayer mod version, its declared BepInEx version, and the same set of other enabled mods. Mixing profiles is the fastest route to a failed join or desync.

How to Install a Megabonk Multiplayer Mod

  1. Close the game and back up first.
    Copy the complete Megabonk save directory before changing the loader or profile.
  2. Install r2modman or Gale from its original page.
    Create a new Megabonk profile named for the package and version, such as Together-5.1.0-test.
  3. Choose exactly one multiplayer package.
    Use the package's Thunderstore manager install option so its declared BepInEx dependency is resolved automatically.
  4. Share the profile specification.
    Every player installs the same multiplayer package version, dependency version and enabled-mod list. A manager profile code is useful, but verify the resolved versions after import.
  5. Launch through the manager.
    Do not launch one player vanilla and another modded. Confirm the new multiplayer or Together menu appears for everyone.
  6. Test a disposable run.
    Verify movement, enemies, XP, chests, Shrines and portal state before using an important save.

Download boundary: use the original Thunderstore links in the comparison table. Avoid reuploads, unknown launchers, password-protected archives and installers that ask for Steam credentials.

Host and Client Requirements

MegabonkTogether

Private room or random queue

  • Host creates a private Friendlies room and shares its code, or players use the random queue.
  • Direct P2P is preferred; relay transport can be used when IPv6 or a failed IPv4 connection requires it.
  • The author describes Windows as the main tested platform. Proton/Steam Deck support is experimental.
BonkWithFriends

Steam friend invite

  • Host selects Multiplayer → Create Multiplayer Game.
  • Host sends a Steam invite, or the client chooses Join Game from the Steam Friends list.
  • Steam must be running and both accounts must be online; the package advertises P2P without port forwarding.

Do not combine projects: MegabonkTogether and BonkWithFriends implement different networking and synchronization layers. Put them in separate profiles if you want to test both.

Compatibility Checklist

Game build

Record the Megabonk version before testing. Launch vanilla once after an official update.

Package and loader

Match the exact multiplayer package and its declared BepInEx IL2CPP dependency on every PC.

Other mods

Start with only the multiplayer package and required libraries. Add cosmetic mods only after a clean test.

Platform

MegabonkTogether 5.0+ documents experimental Proton/Steam Deck cross-play. Do not assume every mod has the same support.

Progression

MegabonkTogether marks shared experience and save progression options as experimental. Keep a backup regardless of the selected mode.

Leaderboard

Do not submit multiplayer or other gameplay-changing mod runs to official leaderboards.

Back Up Saves Before Co-op

MegabonkTogether's author recommends copying the save before installation. With the game closed, back up:

%USERPROFILE%\AppData\LocalLow\Ved\Megabonk
  1. Exit Megabonk and the mod manager completely.
  2. Copy the whole Ved\Megabonk folder to a separate dated location.
  3. Keep the pre-mod backup even after one successful session.
  4. Do not treat Steam Cloud as a versioned backup; a changed save can sync.

Use the Save Recovery Guide before overwriting an active save.

Multiplayer Mod FAQ

Is there a multiplayer mod for Megabonk?

Yes. MegabonkTogether 5.1.0 and BonkWithFriends 0.4.1 were active Thunderstore packages when checked July 24, 2026. Both are unofficial.

Which one should I install?

Choose one. MegabonkTogether offers up to six players, room codes, random queues and an experimental Proton path. BonkWithFriends focuses on Steam friend invites and drop-in/drop-out P2P.

Does every player need the mod?

Yes. Match the multiplayer package, version, BepInEx dependency and all other enabled packages.

Do I need port forwarding?

BonkWithFriends advertises Steamworks P2P without port forwarding. MegabonkTogether uses P2P and provides relay fallback when direct networking fails.

Is multiplayer officially supported?

No. See the official multiplayer status FAQ for the vanilla game.
