Megabonk 300K Kills Guide: Unlock Looooong Top Hat Fast

Turn the 300,000-kill target into a manageable Forest run: choose a scaling build, check whether your clear rate is holding, and fix the run before it stalls.

Quick Answer

To unlock Looooong Top Hat, get 300,000 kills on Forest in a single run. This is not a cumulative grind. Build for sustained map-wide clearing, keep enough survival to protect the run, and use your kill counter rather than the clock alone to decide whether the build is scaling.

What the Run Requires

Required

  • Play on Forest.
  • Reach 300,000 kills.
  • Do it in one run.
  • Keep the run alive until the counter reaches the target.

Not Required

  • No specific character is named in the requirement.
  • No specific weapon is named in the requirement.
  • You do not need to copy a leaderboard loadout exactly.
  • A personal-best time is not part of the unlock condition.
Do not split the target across attempts. A 200K run followed by a 100K run does not satisfy a single-run requirement.

Recommended 300K Kill Build Route

There is no mandatory build. The safest route is to assemble an engine that creates more enemies, levels quickly, reaches across the screen, and survives long enough for those advantages to compound.

  1. Start the XP engine

    Take early upgrades that improve leveling consistency. More levels give you more chances to assemble the full run instead of waiting on a perfect opening.

  2. Add enemy density when your damage can support it

    Cursed Tome is common in historical high-kill runs because more enemies create more kill opportunities. Do not push density ahead of your ability to clear it.

  3. Build overlapping clear zones

    Use weapons that cover different ranges and directions. Flamewalker, Katana, and Dexecutioner appear often in qualifying historical runs; Frostwalker, Firestaff, Axe, and Aura are also common supporting choices.

  4. Protect uptime

    Damage while moving, crowd control, and a survivability layer reduce the time lost to evasive movement. A dead run has zero late-game scaling.

  5. Stop diluting the finished engine

    Once the build clears continuously, strengthen its weakest link. Add damage if enemies accumulate, coverage if they survive off-screen, and defense if movement is becoming erratic.

Why a 300K Attempt Stalls

Enemies are piling up

Your density has outrun damage. Stop increasing pressure and invest in weapon levels, damage, attack frequency, or the range your current setup lacks.

The screen is clear, but kills rise slowly

You may need more enemy density or broader off-center coverage. Add pressure only while the current wave still dies reliably.

You spend too long dodging

Your effective damage uptime is low. Add control or defense, then use weapons that continue clearing while you reposition.

The build scaled early, then flattened

Too many unrelated upgrades can dilute the engine. Reinforce a few weapons and tomes that directly improve growth, density, coverage, and survival.

The run ended above 100K but below 300K

Top Hat and Looooong Top Hat are separate thresholds. Reaching 100K is a useful proof of concept, but the longer unlock still needs 300K in that same run.

The unlock did not appear

First confirm that the run was on Forest and the counter reached at least 300,000 before the run ended. Then check the in-game unlock list and Steam achievement state after returning to the menu.

Leaderboard Strategy Without Chasing a Record

Fast qualifying records are useful because they reveal build patterns, but copying a final screenshot skips the decisions that made the run work. Use leaderboard data in this order:

  1. Look for repeated choices. Noelle and Fox dominate the character sample; Flamewalker, Katana, and Dexecutioner recur across many qualifying runs.
  2. Copy the function, not every slot. Identify which choice supplies growth, density, coverage, control, or survival.
  3. Discard implausible records. Extremely short entries that imply impossible-looking rates should not set your expected pace.
  4. Optimize only after a completion. For the unlock, a stable 300K result matters more than shaving minutes from an attempt.

Frequently Asked Questions

How do I unlock Looooong Top Hat?

Get 300,000 kills on Forest in a single run.

Do kills carry over between runs?

No. The unlock requirement specifies one run, so a new attempt starts the challenge again.

Does the 100K Top Hat unlock count toward 300K?

You may pass the 100K threshold during the same run, but you still need to continue to 300,000 kills for Looooong Top Hat. A previous 100K run does not reduce the later target.

What is the best character for 300K kills?

No character is mandatory. Noelle and Fox appeared most often in the historical qualifying-run sample, making them sensible starting points. Consistent scaling and survival matter more than copying one name.

How long should 300K kills take?

There is no required completion time. Historical leaderboard examples show that some qualifying runs reached the target around 54-60 minutes, but ordinary attempts can take longer. Judge progress by the kill counter.

Should I always take Cursed Tome?

It is common in high-kill runs because added enemy density creates more kill opportunities. Take or upgrade it only while your build can clear the added pressure.
